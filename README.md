# Task Manager App 📋

A React Native app built with Expo for managing tasks.

## Features
- Browse tasks fetched from a REST API
- Save tasks locally on device
- Delete saved tasks
- Dark mode with smooth animation

## Tech Stack
- React Native + Expo
- Expo Router (file-based navigation)
- AsyncStorage (local storage)
- Animated API (dark mode toggle)
- JSONPlaceholder (free mock API)

## Screens
- **Home** — fetches and lists tasks from API
- **Saved** — shows locally saved tasks with delete option
- **Settings** — dark mode toggle with animation

## Run Locally
```bash
npm install
npx expo start
```
